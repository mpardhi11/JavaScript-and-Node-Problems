## Question

Consider the following JavaScript code using a **tagged template literal**.

How is the `highlight` function called, and what will be the output?

---

## Code Snippet

```js
function highlight(strings, ...values) {
  return strings.reduce((result, str, i) => {
    const value = values[i] ? `<em>${values[i]}</em>` : '';
    return result + str + value;
  }, '');
}

const name = 'Sarah';
const score = 95;

// this way of calling the function is a tagged template literal
const message = highlight`Hello ${name}, your score is ${score}!`;
console.log(message);
```

This is a tagged template literal.

JavaScript converts the template into a function call:

```js
// Instead, JavaScript automatically transforms the template literal and calls the function like this:
highlight(
  ['Hello ', ', your score is ', '!'], // strings
  'Sarah', // first value
  95, // second value
);
```

strings receives the text parts.

values receives interpolated expressions.

The function rebuilds the string by wrapping values in <em> tags. The output will be:

```js
Hello <em>Sarah</em>, your score is <em>95</em>!
```

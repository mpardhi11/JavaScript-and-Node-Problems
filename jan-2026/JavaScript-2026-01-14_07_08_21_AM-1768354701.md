## Question

Consider the following JavaScript code that uses **private class fields**, **getter**, and **setter**.

What will be the output of the program?

---

## Code Snippet

```js
class Counter {
  #value = 0; // Private field / Private variable

  // Getter function u can name it anything (but same as setter)
  get count() {
    console.log('Getting count');
    return this.#value;
  }

  // Setter function u can name it anything (but same as getter)
  set count(val) {
    console.log('Setting count to', val);
    this.#value = Math.max(0, val);
  }
}

const counter = new Counter();
counter.count = -5; // Calls the setter
console.log(counter.count); // Calls the getter
```

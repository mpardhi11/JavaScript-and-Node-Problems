# What will be logged to the console?

## JavaScript Closure Stale Value Example

```js
function show() {
  let sum = 0;
  let message = `sum is ${sum}`;

  function print() {
    console.log(message);
  }

  function increase() {
    sum = sum + 10;
  }

  return [print, increase];
}

const [print, increase] = show();
print(); // ?
increase(); // sum changes
print(); // ?
```

**Short interview-style answer:**

**Output:**

sum is 0 sum is 0

**Why (1â€“2 lines):** `message` is created once when `show()` runs. `increase()` updates `sum`, but `message` stays the
same, so `print()` logs a **stale value**.

**Fix (if asked):**

```js
function print() {
  console.log(`sum is ${sum}`);
}
```

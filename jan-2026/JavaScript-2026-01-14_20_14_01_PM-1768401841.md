# JavaScript Interview Question: Remove Duplicate Characters from a String

## Question

You are given a string.  
Your task is to remove duplicate characters from the string.

Implement the solution for **two scenarios**:

1. **Case 1:** Ignore case sensitivity
2. **Case 2:** Consider case sensitivity

You must preserve the **original order** of characters.

---

## Scenario 1: Ignoring Case Sensitivity

### Problem Statement

Remove duplicate characters from a string **without considering case sensitivity**. This means characters like `'H'` and
`'h'` should be treated as the **same**.

### Example

**Input:**

```text
"HeLLo"
```

**Output:**

```text
"helo"
```

---

### Answer (Code)

```js
function removeDuplicateIgnoreCase(str) {
  const seen = {};
  let result = '';

  for (const char of str.toLowerCase()) {
    if (!seen[char]) {
      result += char;
      seen[char] = true;
    }
  }
  return result;
}

console.log(removeDuplicateIgnoreCase('HeLLo'));
```

### Explanation

- Convert the string to lowercase using `toLowerCase()`
- Track characters using an object
- Append only unseen characters to the result
- Preserves character order

---

## Scenario 2: Considering Case Sensitivity

### Problem Statement

Remove duplicate characters from a string **while considering case sensitivity**. This means characters like `'H'` and
`'h'` are treated as **different** characters.

### Example

**Input:**

```text
"HeLLo"
```

**Output:**

```text
"HeLLo"
```

---

### Answer (Code)

```js
function removeDuplicateWithCase(str) {
  const seen = {};
  let result = '';

  for (const char of str) {
    if (!seen[char]) {
      result += char;
      seen[char] = true;
    }
  }
  return result;
}

console.log(removeDuplicateWithCase('HeLLo'));
```
